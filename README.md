## About The Project

BMS1 is a comprehensive Battery Management System designed to optimize battery performance, ensure safety, and extend battery life across various applications. This project provides robust tools for monitoring, control, and analysis, enabling efficient power management and reliable operation of battery packs.

### Key Features
-   **Real-time Cell Monitoring:** Track individual cell voltages, currents, and temperatures to detect anomalies quickly.
-   **State-of-Charge (SoC) & State-of-Health (SoH) Estimation:** Accurate algorithms to provide insights into battery capacity and overall degradation.
-   **Protection Mechanisms:** Implement safeguards against overcharge, over-discharge, over-current, and extreme temperatures, ensuring battery longevity and user safety.
-   **Cell Balancing:** Actively manage cell voltage disparities to maximize usable capacity and prevent premature aging of individual cells.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Before you begin, ensure you have the following installed:
*   `git`
*   A C++ compiler (e.g., GCC, Clang)
*   Python 3.x
*   `pip` (Python package installer)

### Installation
1.  Clone the repository:
    ```sh
    git clone https://github.com/krishujeniya/BMS1.git
    ```
2.  Navigate into the project directory:
    ```sh
    cd BMS1
    ```
3.  (Optional) Install Python dependencies:
    ```sh
    pip install -r requirements.txt
    ```
